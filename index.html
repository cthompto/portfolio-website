New Work:    Consensus Portraits


Compressed version of Consensus Portrait #1, 2022



Consensus Portraits turns the algorithmic gaze of machine learning platforms on one another as a way of exploring the inherent biases of these platforms. Using machine learning algorithms that generate, animate, and analyze human faces, this project considers how these platforms generate consensus on what is and is not human.
